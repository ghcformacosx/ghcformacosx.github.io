GHC for Mac OS X GitHub Projects

Install Haskell on Mac without any fuss

  • Download and unzip ghc-7.8.2-r1.zip
  • Move it to a stable location such as /Applications
  • Double-click for instructions to add it to your Terminal's $PATH

Requirements

  • Xcode 5
  • Only tested on 64-bit Mac OS X 10.9

What it includes

  • GHC 7.8.2
  • cabal-install 1.20.0.1
  • HTML documentation
  • man pages
  • A small GUI app wrapper to help you set up your $PATH
